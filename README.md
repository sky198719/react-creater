react16-webpack4-redux脚手架说明
css预处理：less
单页面路由：react-router
状态管理机：react-redux

文件路径
index.js 入口文件
css 公用css文件
js 公用js文件
js/action redux的action文件
js/reducer redux的reducer文件
js/store redux的stroe文件
mods 业务组件
mods/... 多页面文件夹
mods/.../index.js 多页面接入层
mods/.../views 多页面组件
mui 公用组件
template 模板

启动本地服务
npm run server

本地打包
npm run build
